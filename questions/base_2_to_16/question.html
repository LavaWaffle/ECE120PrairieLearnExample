Write a program that converts from base 2 to base 16

The program will read one input.

  1. The unsigned integer N in base 2

The program expects one output: The integer N in base 16.

Constraints: N will be between $00000000_2$ ($0_{16}$) and $11111111_2$ ($FF_{16}$). N will always only contain 0s and 1s. N will always have 8 bits.

Example input/output:

  • Input: 00000101
  • Output: 05
Converting $00000101_2$ to base 16 gives $05_{16}$.

Example input/output:

  • Input: 00001101
  • Output: 0D
Converting $00001101_2$ to base 16 gives $0D_{16}$.

The %X format specifier prints the hexadecimal representation of an integer.
