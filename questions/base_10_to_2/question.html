Write a program to convert from base 10 to base 2.

The program will read one input.

  1. The unsigned integer N in base 10

The program expects one output: The integer N in base 2.

Note: N will be between 0 (0 in base 2) and 63 (111111 in base 2).

Example input/output:

  • Input: 5
  • Output: 101
Converting 5 to base 2 gives 101.

Example input/output:

  • Input: 10
  • Output: 1010
Converting 10 to base 2 gives 1010.

Example input/output:

  • Input: 32
  • Output: 100000
Converting 32 to base 2 gives 100000.

Example input/output:

  • Input: 63
  • Output: 111111
Converting 63 to base 2 gives 111111.
