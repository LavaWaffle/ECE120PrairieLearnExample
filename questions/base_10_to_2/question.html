Write a program to convert from base 10 to base 2.

The program will read one input.

  1. The unsigned integer N in base 10

The program expects one output: The integer N in base 2.

Constraint 1: N will be between $0_{10}$ ($0_{2}$) and $63_{10}$ ($111111_{2}$).

Constraint 2: Your output must always have 6 bits.

Example input/output:

  • Input: 5
  • Output: 000101
Converting $5_{10}$ to base 2 gives $000101_{2}$.

Example input/output:

  • Input: 10
  • Output: 001010
Converting $10_{10}$ to base 2 gives $001010_{2}$.

Example input/output:

  • Input: 32
  • Output: 100000
Converting $32_{10}$ to base 2 gives $100000_{2}$.

Example input/output:

  • Input: 63
  • Output: 111111
Converting $63_{10}$ to base 2 gives $111111_{2}$.

When converting to base 2, we can do the "will it fit" method. We begin by seeing if the number is bigger than the 32 (because we know that 0 < N < 64. Thus the highest bit is the 6th bit: 2^5 = 32). If it is output a 1. Else output a 0.
