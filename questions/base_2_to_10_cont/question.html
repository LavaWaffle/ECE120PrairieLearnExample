Write a program that converts from base 2 to base 10. The input is going to be of length L.

The program will read two inputs .

  1. The unsigned integer L: the length of N
  2. The unsigned integer N in base 2

The program expects one output: The integer N in base 10.

Constraints: N will always be of length L. L will be between $1_{10}$ and $16_{10}$.

Example input/output:

  • Input: 10 0000000101
  • Output: 5
Converting $0000000101_2$ to base 10 gives $5_{10}$.

Example input/output:

  • Input: 4 1101
  • Output: 13
Converting $0000001101_2$ to base 10 gives $13_{10}$.

1 << (L - 1) is equivalent to 2 ^ (L - 1) (aka the value of the MSB in a L bit number)
