In ECE 120 we often convert between base 2 and base 10 by hand. However, computers do this conversion all the time. Write a program that converts a number in base 2 to base 10.

The program will read one input.

  1. The unsigned integer N in base 2

The program expects one output: The integer N in base 10.

Note: N will be between 0 (0) and 111111 (63 in base 10). N will always only contain 0s and 1s. N will always have 6 bits.

Example input/output:

  • Input: 000101
  • Output: 5
Converting 000101 to base 10 gives 5.

Example input/output:

  • Input: 001101
  • Output: 13
Converting 001101 to base 10 gives 13.

1 << 5 is equivalent to 2 ^ 5 (aka the value of the MSB in a 6 bit number) power = power >> 1 shifts the power variable right by 1 (dividing by 2)
