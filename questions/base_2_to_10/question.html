In ECE 120 we often convert between base 2 and base 10 by hand. However, computers do this conversion all the time. Write a program that converts a number in base 2 to base 10.

The program will read one input.

  1. The unsigned integer N in base 2

The program expects one output: The integer N in base 10.

Note: N will be between 0 (0) and 111111 (63 in base 10). N will always only contain 0s and 1s.

Example input/output:

  • Input: 101
  • Output: 5
Converting 101 to base 10 gives 5.

Example input/output:

  • Input: 1101
  • Output: 13
Converting 1101 to base 10 gives 13.

2^x is the same as 1 << x N % 10 gives the last digit in base 2
Ex: 101 % 10 = 1
Ex: 1100 % 10 = 0 N /= 10 removes the last digit of N in base 2
