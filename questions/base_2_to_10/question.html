In ECE 120 we often convert from base 2 to base 10 by hand. However, computers do this conversion all the time. Lets write a program that converts a number in base 2 to base 10.

The program will read one input.

  1. The unsigned integer N in base 2

The program expects one output: The integer N in base 10.

Constraints: N will be between $000000_2$ ($0_{10}$) and $111111_2$ ($63_{10}$). N will always only contain 0s and 1s. N will always have 6 bits.

Example input/output:

  • Input: 000101
  • Output: 5
Converting $000101_2$ to base 10 gives $5_{10}$.

Example input/output:

  • Input: 001101
  • Output: 13
Converting $001101_2$ to base 10 gives $13_{10}$.
